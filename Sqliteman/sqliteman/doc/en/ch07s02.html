Create Table

The (columns), AS query, and EDIT SQL tabs allow you to create a table in various ways.

Database

A combo box with databases attached. Select one and the table will be created in this schema.

Table Name

Name of new table. It must be nonempty.

(columns) tab

Use "Add" and "Remove" buttons to create columns for this table. Every column must have a table-unique name, chosen data type (prepared combo box), optional NOT NULL constraint, and optional DEFAULT value. If you want to use your own non-standard data type, you can type it into the combo box.

Important

Consult Sqlite documentation for data type, NOT NULL, and DEFAULT handling specials, please.

AS query tab

This opens the query builder where you can create a table using a CREATE TABLE name AS SELECT ... statement.

Edit SQL tab

You can write any SQL CREATE TABLE statement here. The first time you open this tab, it will make a CREATE TABLE statement from the information in the previously open tab. Every subsequent time you open this tab you will be asked if you want to keep the existing SQL or make it from the information in the previously open tab again.

Create

This button is disabled when the (columns) tab is selected unless the new table and all columns have nonempty names, and disabled when the AS query tab is selected unless the new table has a nonempty name. Press it to create the table. The result of this operation is shown in the text area below, which will expand if necessary to contain a long error message.
