How to obtain Sqliteman

Source Code

For the sources of the latest version go to github, click on the Clone or Download button and select "Download ZIP".

There is also a prebuilt binary (generally a bit out of date with respect to the sources) in github here.

If you find a prebuilt binary elsewhere, Help-> About... will show you its version. The current version of the sorce code is here.

MS Windows

Unfortunately I have no access to any MS Windows machine. You can find the sources for this version in the Sqliteman area on github. You should be able to build it.

Warning

Downloadable Sqliteman for MS Windows can be found on the various third party sites. You can not be sure what these packages contain. Everything is propably fine there but there is no content warranty of 3rd party packages and they may be older versions.

Linux

It's highly probable you have Sqliteman binary package prepared for your distribution. However this may have been built from old and no longer maintained sources at pvanek/sqliteman. It should be better to build from the sources in github here.

Mac OS X

Unfortunately I have no access to any Mac machine. Sqliteman should compile and run on this platform. It may be provided by 3rd parties again, but you are probably better off building from the sources in github here if you can.

Other UNIXes

Some reports from users tell that it can be setup on Solaris, Opensolaris, *BSD. It may be provided by 3rd parties again, but you are probably better off building from the sources in github here if you can.
